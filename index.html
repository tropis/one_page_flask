Objective

A one-page Flask program is useful for learning, extending, and for hacking together a new project. This one actually combines a few technologies and can run on your computer:

  • Python 3
  • Virtual environment for Python
  • Flask application with 3 routes and the built-in Template package
  • Bulma CSS Framework
  • Google Fonts
  • Flask built-in webserver for local testing

Bulma is similar to Bootstrap but smaller and easier to understand. It is well documented.

Template

I said this is one page Flask and one page template. There is actually a second version that doesn't use the template file. A tiny layout is included at the bottom of index.html, but it was so boring that I made it optional. Change the MINI parameter in the program.
