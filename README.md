# one_page_flask

A simple flask application in one page with Bulma css framework.

## Installation

Download zip file, it unzips into a new directory.
Or clone.

## Running

Run in local shell.

```shell
pip install flask
python flask_one_page.py
```
Point browser to http://127.0.0.1:5000
